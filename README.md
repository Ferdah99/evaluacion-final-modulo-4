# Administrador de Usuarios — Evaluación Final Módulo 4

## Descripción

Aplicación web que consume el servicio [JSONPlaceholder](https://jsonplaceholder.typicode.com/users) para administrar información de usuarios. La data se obtiene mediante `XMLHttpRequest` y se gestiona a través de una clase JavaScript (`UserManager`).

## Objetivos

- Consumir un Web Service externo usando `XMLHttpRequest`.
- Encapsular la lógica de datos dentro de una clase JavaScript.
- Exponer métodos que permitan consultar la información de los usuarios de distintas formas:
  - Listar todos los nombres.
  - Buscar información básica (username y correo) por nombre.
  - Consultar la dirección completa de un usuario por nombre.
  - Consultar información avanzada (teléfono, sitio web y compañía) por nombre.
  - Listar todas las compañías con su *catchphrase*.
  - Listar nombres ordenados alfabéticamente.
- Integrar cada método con un botón HTML independiente.

## Tecnologías

- HTML5
- CSS3
- JavaScript (ES6 — clases, arrow functions, destructuring)
- XMLHttpRequest

## Uso

1. Abre `index.html` en el navegador.
2. Espera a que el indicador inferior muestre **"✓ 10 usuarios cargados"**.
3. Abre la consola del navegador (F12 → Console).
4. Haz clic en cualquiera de los seis botones para ver los resultados en consola.
